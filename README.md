## How do i create my own Distrobution?
Step 1: Download the Source Code
Step 2: Modify the Code to your liking

If you're ready to build your Distro, follow the steps below!
Step 1: Startup your Distro
Step 2: Open the Terminal within your Distro
Step 3: Execute the Command "buildpios"
Your Distro should be ready soon!
